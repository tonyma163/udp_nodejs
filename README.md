# udp_nodejs

Nodejs UDP Simple Chat Room

Using 'dgram' & 'readline'

# Install
git clone https://github.com/tonyma163/udp_nodejs
cd app
npm i

# Run the server
node server.js

# Run the client
node client.js

Modify the 'client.js' roomId to change the default entering/sending message room.
